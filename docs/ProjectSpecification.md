| 항목              | 내용                                                                                                                                             |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |

- 저사양 디바이스(Raspberry Pi 3)에서 AI 모델을 실행 가능한 수준으로 최적화

- 경량화 기법(양자화) 적용 전/후 성능 변화 정량적 분석
- 전체 ML 파이프라인(데이터 → 학습 → 최적화 → 배포) 경험 확보

## 3. 실험 대상 모델

### 모델: MediaPipe PoseLandmarker (Lite)

| 항목            | 내용                                                  |
| --------------- | ----------------------------------------------------- |
| **특징**        | 인체 33개 관절 포인트를 실시간으로 추정하는 경량 모델 |
| **입력 데이터** | 카메라 영상 또는 샘플 영상                            |
| **출력 데이터** | 인체 33개 관절 좌표, FPS                              |

**선정 이유**

- 라즈베리파이 환경에서 공식 동작 사례 존재 (MediaPipe 공식 예제)
- TFLite 기반으로 양자화 파이프라인 적용이 용이
- FPS, Latency 등 정량적 비교가 명확함

## 4. 적용 기술 및 파이프라인

### 4.1 전체 파이프라인

```
공개 데이터셋 수집 (COCO 등)
        ↓
데이터 증강 (Albumentations)
        ↓
고품질 데이터 선별
        ↓
파인튜닝 (MediaPipe Model Maker / Google Colab)
        ↓
.tflite 모델 추출
        ↓
ONNX 변환 (tensorflow-onnx)
        ↓
모델 최적화 (onnx-simplifier)
        ↓
Int8 양자화 (block_quantize)
        ↓
라즈베리파이 배포 (ONNX Runtime)
```

### 4.2 데이터 처리

| 항목            | 내용                                                              |
| --------------- | ----------------------------------------------------------------- |
| **데이터셋**    | COCO, Human3.6M 등 공개 포즈 데이터셋                             |
| **데이터 증강** | Albumentations 라이브러리 (회전, 플립, 밝기 조절, 노이즈 추가 등) |
| **데이터 선별** | 관절 가시성 기준으로 고품질 데이터 필터링                         |

### 4.3 모델 경량화

**파인튜닝**

- MediaPipe Model Maker를 활용한 커스텀 데이터 재학습
- 실행 환경: Google Colab (T4 GPU)

**양자화 (Quantization)**

- Float32 → Int8 변환
- block_quantize.py 활용 (block_size=64)
- 예상 효과: 모델 크기 ¼ 축소, 추론 속도 향상

### 4.4 실행 환경

| 구분           | 환경                                     |
| -------------- | ---------------------------------------- |
| **학습 환경**  | Google Colab (T4 GPU)                    |
| **배포 환경**  | Raspberry Pi 3 (64비트 OS)               |
| **프레임워크** | MediaPipe, TensorFlow Lite, ONNX Runtime |
| **언어**       | Python                                   |

## 5. 성능 평가 방법

### 5.1 평가 지표

| 지표             | 설명                                  |
| ---------------- | ------------------------------------- |
| **FPS**          | 실시간 처리 가능 여부 판단            |
| **Latency**      | 프레임 입력 → 결과 출력까지 지연 시간 |
| **Accuracy**     | 관절 위치 오차 (PCK 기준)             |
| **Memory Usage** | 임베디드 환경 적합성 판단             |
| **Model Size**   | 양자화 전/후 모델 파일 크기 비교      |

### 5.2 실험 방식

동일한 Raspberry Pi 3 환경에서 3가지 모델 비교:

| 모델           | 설명                               |
| -------------- | ---------------------------------- |
| **Baseline**   | MediaPipe 공식 원본 모델 (Float32) |
| **Fine-tuned** | 파인튜닝 적용 모델                 |
| **Quantized**  | 파인튜닝 + Int8 양자화 모델        |
